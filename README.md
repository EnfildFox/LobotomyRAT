[ПРИМЕЧАНИЯ]
• Все команды чувствительны к регистру? Нет. Парсинг использует точное совпадение или .find().
• Неизвестная команда → возвращает "UNKNOWN_COMMAND".
• При активном анти-детекте (g_sleepMode == true) → все команды возвращают "SLEEP_MODE".
• Заглушки возвращают строки, но НЕ прерывают соединение. Агент остаётся на линии.
